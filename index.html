Tagged.prototype.convert

convert(converters: Object): any

If we have a converter for this type, do the conversion. Some converters are built-in. Additional ones can be passed in. If you want to remove a built-in converter, pass a converter in whose value is 'null' instead of a function.

Parameters

  1. converters: Object:  
    keys in the object are a tag number, the value is a function that takes the decoded CBOR and returns a JavaScript value of the appropriate type. Throw an exception in the function on errors.

Returns

any the converted item
